# Tutorial 1: Learn Motion Graphics in 10 Minutes

**Based on:** "Learn After Effects in 10 Minutes" by Ben Marriott
**Source:** youtube.com/watch?v=hBEHRD89bZI

---

## What You'll Learn

- Creating compositions
- Importing and organizing assets
- Animating with keyframes
- Using easing and the curve editor
- Creating text layers
- Exporting your work

---

## 1. Project Setup

1. **Launch Weyl** in ComfyUI sidebar
2. **Import media** - `Ctrl+I` or drag files to Project Panel
3. **Organize** - Create folders in Project Panel as needed

---

## 2. Create Composition

1. Press `Ctrl+K` for Composition Settings
2. Set **Name**, **Width/Height**, **Frame Rate** (16fps for AI), **Duration**
3. Click OK - Timeline and Canvas appear

---

## 3. Add Assets to Timeline

1. **Drag** assets from Project Panel to Timeline
2. Layers stack top-to-bottom (top renders on top)
3. **Reorder** by dragging layers in the Timeline

---

## 4. Playback & Navigation

| Action | Shortcut |
|--------|----------|
| Play/Pause | `Space` |
| Go to start | `Home` |
| Go to end | `End` |
| Step +1 frame | `→` |
| Step -1 frame | `←` |
| Step +10 frames | `Shift+→` |

---

## 5. Transform Properties

Select a layer, then press:

| Key | Property |
|-----|----------|
| `P` | Position |
| `S` | Scale |
| `R` | Rotation |
| `T` | Opacity |
| `A` | Origin (Anchor) |
| `U` | Show all animated properties |

**Change values** by:
- Dragging in Properties Panel
- Typing numbers directly
- Dragging layer in Canvas

---

## 6. Animate with Keyframes

1. Move playhead to frame 0
2. Select layer, press `P` for Position
3. Click the **stopwatch** (◆) to enable animation
4. Move playhead to 1 second
5. Change Position value
6. Keyframe auto-creates
7. Press `Space` to preview

**Repeat** for Scale, Rotation, Opacity as needed.

---

## 7. Add Easing

1. Select keyframes
2. Press `F9` for Smooth easing (or right-click → Ease)
3. Press `Shift+G` to open Curve Editor
4. Drag handles to adjust motion curve
5. Press `Shift+G` to close

| Shortcut | Effect |
|----------|--------|
| `F9` | Ease In/Out |
| `Shift+F9` | Ease In |
| `Ctrl+Shift+F9` | Ease Out |

---

## 8. Create Text Layer

1. Press `T` for Text tool (or use Create menu)
2. Click in Canvas
3. Type your text
4. Adjust in Properties Panel:
   - Font family/size
   - Color
   - Alignment
5. Animate Position/Opacity as with any layer

---

## 9. Align Layers

1. Select layer(s)
2. Open Align panel (right sidebar)
3. Click alignment buttons:
   - Horizontal: Left, Center, Right
   - Vertical: Top, Center, Bottom

---

## 10. Add Effects

1. Select layer
2. Open Effects panel (right sidebar)
3. Click **Add Effect** dropdown
4. Choose effect (Blur, Glow, Color Correction, etc.)
5. Adjust parameters
6. Drag to reorder effects

---

## 11. Nested Compositions

1. Select layers to group
2. Press `Ctrl+Shift+C` for Pre-compose
3. Enter name
4. Nested comp appears as single layer
5. Double-click to edit inside

---

## 12. Export

1. Press `Ctrl+M` for Export dialog
2. Choose format:
   - **PNG Sequence** - for AI workflows
   - **MP4** - for video
3. Set frame range
4. Click Export

---
