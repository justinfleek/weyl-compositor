# Tutorial 2: 50 Motion Graphics Productivity Tips

**Based on:** "50 After Effects Tips" by Ben Marriott
**Goal:** Work faster with keyboard shortcuts and workflow tricks

---

## Timeline Navigation

| Tip | Shortcut | Action |
|-----|----------|--------|
| 1 | `I` | Jump to layer In Point |
| 2 | `O` | Jump to layer Out Point |
| 3 | `[` | Move layer start to playhead |
| 4 | `]` | Move layer end to playhead |
| 5 | `Alt+[` | Trim layer start to playhead |
| 6 | `Alt+]` | Trim layer end to playhead |
| 7 | `J` | Previous keyframe |
| 8 | `K` | Next keyframe |
| 9 | `Ctrl+↑/↓` | Select layer above/below |
| 10 | `Home/End` | Go to start/end |

---

## Property Shortcuts

Select layer, then press:

| Tip | Key | Property |
|-----|-----|----------|
| 11 | `P` | Position |
| 12 | `S` | Scale |
| 13 | `R` | Rotation |
| 14 | `T` | Opacity |
| 15 | `A` | Origin (Anchor) |
| 16 | `U` | All animated properties |
| 17 | `UU` | All modified properties |
| 18 | `E` | Effects |
| 19 | `EE` | Expressions |
| 20 | `M` | Masks |

**Add to current:** Hold `Shift` + key

## Pro Tips

1. **Double-click** layer to open nested comp
2. **Right-click** keyframe for interpolation menu
3. **Drag handles** in curve editor for custom easing
4. **Marquee select** multiple keyframes on timeline
5. **Alt+drag** to duplicate layer while moving
